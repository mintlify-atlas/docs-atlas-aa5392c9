## What is Walty?

Walty is a free and open-source self-custodial wallet for EVM networks. It helps you manage crypto assets across Ethereum-compatible networks from a single interface with complete control over your private keys.

The wallet follows a self-custody model: seed material is encrypted locally in the browser and transaction signing happens client-side. No tracking, no analytics, and fully self-hostable.

## Supported Networks

Walty supports all major EVM-compatible networks:

- **Ethereum** - The original smart contract platform
- **Arbitrum** - Layer 2 scaling solution for Ethereum
- **Base** - Coinbase's Layer 2 network
- **Optimism** - Optimistic rollup for Ethereum
- **Polygon** - Sidechain for fast, low-cost transactions

## Quick Example

```bash
# Clone the repository
git clone https://github.com/ignaciogarcia-dev/walty.git
cd walty

# Copy environment variables
cp .env.example .env

# Start with Docker Compose
docker compose up --build

# Access at http://localhost:3000
```
