---
title: 'Get Transaction History'
api: 'GET /api/tx'
description: 'Retrieve transaction history for the authenticated user'
---

## Authentication

This endpoint requires authentication. Include your session token in the request headers.

## Query Parameters

<ParamField query="limit" type="number" default={20}>
  Number of transactions to return (max 100)
</ParamField>

<ParamField query="offset" type="number" default={0}>
  Number of transactions to skip for pagination
</ParamField>

## Response

Returns an array of transaction objects, ordered by creation date (newest first).

<ResponseField name="id" type="number">
  Unique transaction ID
</ResponseField>

<ResponseField name="userId" type="string">
  User ID who owns this transaction
</ResponseField>

<ResponseField name="hash" type="string">
  Transaction hash
</ResponseField>

<ResponseField name="chainId" type="number">
  Chain ID where the transaction occurred
</ResponseField>

<ResponseField name="chainType" type="string">
  Type of blockchain (e.g., "EVM")
</ResponseField>

<ResponseField name="fromAddress" type="string">
  Sender's wallet address
</ResponseField>

<ResponseField name="toAddress" type="string">
  Recipient's wallet address
</ResponseField>

<ResponseField name="tokenAddress" type="string | null">
  Token contract address (null for native tokens)
</ResponseField>

<ResponseField name="tokenSymbol" type="string">
  Token symbol
</ResponseField>

<ResponseField name="value" type="string">
  Transaction amount
</ResponseField>

<ResponseField name="status" type="string">
  Transaction status: `"pending"`, `"confirmed"`, or `"failed"`
</ResponseField>

<ResponseField name="createdAt" type="string">
  ISO 8601 timestamp when the transaction was recorded
</ResponseField>

## Example Request

<CodeGroup>
```bash cURL
curl -X GET "https://api.walty.com/api/tx?limit=10&offset=0" \
  -H "Cookie: session=your_session_token"
```

```javascript JavaScript
const response = await fetch(
  'https://api.walty.com/api/tx?limit=10&offset=0',
  {
    headers: {
      'Cookie': 'session=your_session_token'
    }
  }
);
const transactions = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://api.walty.com/api/tx',
    params={'limit': 10, 'offset': 0},
    cookies={'session': 'your_session_token'}
)
transactions = response.json()
```
</CodeGroup>

## Error Responses

<ResponseField name="error" type="string">
  Error message
</ResponseField>

```json 401 Unauthorized
{
  "error": "Unauthorized"
}
```

## Pagination

Use `limit` and `offset` parameters to paginate through results:

- First page: `?limit=20&offset=0`
- Second page: `?limit=20&offset=20`
- Third page: `?limit=20&offset=40`

<Note>
  Transactions are sorted by creation date in descending order (newest first). The maximum limit per request is 100.
</Note>
