## Overview

Swap execution in Walty is a **client-side operation**. There are no separate `/api/swap/approve` or `/api/swap/execute` endpoints. Instead, you execute swaps directly on-chain using the transaction data returned from the [Get Swap Quote](/api/swaps/quote) endpoint.

## Execution Flow

### 1. Get a Quote

First, fetch a swap quote:

```javascript
const params = new URLSearchParams({
  sellToken: '0xA0b86991c6218b36c1d19D4a2e9Eb0cE3606eB48', // USDC
  buyToken: 'ETH',
  sellAmount: '1000000000', // 1000 USDC
  takerAddress: userAddress,
  chainId: '1'
});

const response = await fetch(
  `https://api.walty.com/api/swap/quote?${params}`,
  { headers: { 'Cookie': 'session=your_session_token' } }
);
const quote = await response.json();
```

### 2. Check for Allowance Issues

If the quote includes an allowance issue, approve token spending first:

```javascript
if (quote.issues?.allowance) {
  // ERC20 token approval required
  const tokenContract = new ethers.Contract(
    sellTokenAddress,
    ['function approve(address spender, uint256 amount) returns (bool)'],
    signer
  );
  
  const approveTx = await tokenContract.approve(
    quote.allowanceTarget,
    quote.sellAmount
  );
  
  // Wait for approval to be mined
  await approveTx.wait();
  
  console.log('Token approval confirmed');
}
```

### 3. Execute the Swap

Send the transaction using the data from the quote:

```javascript
const swapTx = await signer.sendTransaction({
  to: quote.transaction.to,
  data: quote.transaction.data,
  value: quote.transaction.value,
  gasLimit: quote.transaction.gas
});

// Wait for transaction to be mined
const receipt = await swapTx.wait();
console.log('Swap executed:', receipt.transactionHash);
```

### 4. Record the Transaction (Optional)

Record the swap in Walty's database for tracking:

```javascript
await fetch('https://api.walty.com/api/tx', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Cookie': 'session=your_session_token'
  },
  body: JSON.stringify({
    hash: receipt.transactionHash,
    chainId: 1,
    chainType: 'EVM',
    fromAddress: userAddress,
    toAddress: quote.transaction.to,
    tokenAddress: sellTokenAddress,
    tokenSymbol: 'USDC',
    value: ethers.formatUnits(quote.sellAmount, 6)
  })
});
```

import { ethers } from 'ethers';

async function executeSwap({
  sellToken,
  buyToken,
  sellAmount,
  userAddress,
  signer,
  chainId = 1
}) {
  // 1. Get quote
  const params = new URLSearchParams({
    sellToken,
    buyToken,
    sellAmount,
    takerAddress: userAddress,
    chainId: chainId.toString()
  });
  
  const quoteResponse = await fetch(
    `https://api.walty.com/api/swap/quote?${params}`,
    { headers: { 'Cookie': 'session=your_session_token' } }
  );
  
  if (!quoteResponse.ok) {
    throw new Error('Failed to get quote');
  }
  
  const quote = await quoteResponse.json();
  
  // 2. Handle token approval if needed
  if (quote.issues?.allowance) {
    console.log('Approving token spending...');
    
    const tokenContract = new ethers.Contract(
      sellToken,
      ['function approve(address spender, uint256 amount) returns (bool)'],
      signer
    );
    
    const approveTx = await tokenContract.approve(
      quote.allowanceTarget,
      quote.sellAmount
    );
    
    await approveTx.wait();
    console.log('Approval confirmed');
  }
  
  // 3. Execute swap
  console.log('Executing swap...');
  const swapTx = await signer.sendTransaction({
    to: quote.transaction.to,
    data: quote.transaction.data,
    value: quote.transaction.value,
    gasLimit: quote.transaction.gas
  });
  
  const receipt = await swapTx.wait();
  console.log('Swap successful:', receipt.transactionHash);
  
  return receipt;
}

import { createWalletClient, http, parseUnits } from 'viem';

async function executeSwap({
  sellToken,
  buyToken,
  sellAmount,
  userAddress,
  walletClient,
  chainId = 1

  // 1. Get quote
  const params = new URLSearchParams({
    sellToken,
    buyToken,
    sellAmount,
    takerAddress: userAddress,
    chainId: chainId.toString()
  });
  
  const quoteResponse = await fetch(

    { headers: { 'Cookie': 'session=your_session_token' } }

  const quote = await quoteResponse.json();
  
  // 2. Handle token approval if needed
  if (quote.issues?.allowance) {
    const hash = await walletClient.writeContract({
      address: sellToken,
      abi: [{
        name: 'approve',
        type: 'function',
        inputs: [
          { name: 'spender', type: 'address' },
          { name: 'amount', type: 'uint256' }
        ],
        outputs: [{ type: 'bool' }]
      }],
      functionName: 'approve',
      args: [quote.allowanceTarget, BigInt(quote.sellAmount)]
    });
    
    // Wait for approval
    await walletClient.waitForTransactionReceipt({ hash });
  }
  
  // 3. Execute swap
  const hash = await walletClient.sendTransaction({
    to: quote.transaction.to,
    data: quote.transaction.data,
    value: BigInt(quote.transaction.value),
    gas: BigInt(quote.transaction.gas)
  });
  
  const receipt = await walletClient.waitForTransactionReceipt({ hash });
  return receipt;
}

## Transaction Object Structure

The quote response includes a `transaction` object ready for execution:

<ParamField body="to" type="string">
  DEX aggregator contract address (0x or 1inch router)
</ParamField>

<ParamField body="data" type="string">
  Encoded function call data (includes swap parameters and routing)
</ParamField>

<ParamField body="value" type="string">
  ETH amount to send (in wei) - only non-zero for native token swaps
</ParamField>

<ParamField body="gas" type="string">
  Estimated gas limit for the transaction
</ParamField>

<ParamField body="gasPrice" type="string">
  Suggested gas price in wei
</ParamField>

## Handling Issues

### Allowance Issue

<Note>
  If `quote.issues.allowance` exists, the DEX aggregator needs permission to spend your tokens. You must approve the `allowanceTarget` address before executing the swap.
</Note>

```javascript
if (quote.issues?.allowance) {
  // Current allowance is insufficient
  // Approve the allowanceTarget to spend sellAmount
}
```

### Balance Issue

<Warning>
  If `quote.issues.balance` exists, the user doesn't have enough tokens. The swap will fail if you attempt to execute it.
</Warning>

```javascript
if (quote.issues?.balance) {
  throw new Error(
    `Insufficient balance: have ${quote.issues.balance.actual}, need ${quote.issues.balance.expected}`
  );
}
```

## Best Practices

1. **Always check for issues** before executing the swap
2. **Wait for approval transaction** to be mined before executing the swap
3. **Set appropriate gas limits** - use the provided `gas` value or add a buffer
4. **Handle transaction failures** - monitor the transaction status on-chain
5. **Record the transaction** in Walty's database for tracking and activity history
